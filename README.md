# 🚗 Car Service Test Database

Простая консольная база данных на **Python** для автоматизации процессов в автосервисе. Проект создан для демонстрации навыков работы с системами контроля версий Git и базовой логики программирования.

---

## 🛠 Технологии
В проекте используются следующие инструменты:
* **Язык программирования:** Python 3.x
* **Система контроля версий:** Git
* **Облачный репозиторий:** GitHub

---

## 🚀 Возможности программы
На данный момент проект находится в активной разработке. Текущая версия (v1.1) включает в себя:
1. **Учет автомобилей:** Базовый функционал для работы с базой данных машин.
2. **Расчет стоимости:** Логика вычисления стоимости ремонта и обслуживания.
3. **Расширенный функционал:** Экспериментальный модуль для обслуживания спецтехники и воздушного транспорта (вертолётов 🛸).

---

## 💻 Как запустить проект локально

1. Склонируйте репозиторий к себе на компьютер:
   ```bash
   git clone https://github.com/zeldeck/car-service-test.git
   ```

2. Перейдите в папку с проектом:
   ```bash
   cd car-service-test
   ```

3. Запустите главный файл программы через терминал:
   ```bash
   python car2.py
   ```
